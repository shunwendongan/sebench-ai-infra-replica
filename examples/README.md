# Examples

- `requirements.json`: public-paper reproduction requirement used by the authoring agent.
- `golden_tasks.json`: fixed dataset fixture for regression tests.
- `toy_benchmark.json`: three true-loop toy tasks that exercise Work/Judge separation,
  patch hashing, hidden judge rules, metrics, and failure typing.
- `synthetic_spatial_scene.json`: a tiny 3D transform fixture with known translation and rotation.
